# Ravya-Mental-Health-Assistant
Ravya - Mental Health Assistant

## Installation

Clone the project

```bash
  git clone https://github.com/Humanshere/Ravya-Mental-Health-Assistant
```

Go to the project directory

```bash
  cd Ravya-Mental-Health-Assistant
```

Install dependencies

```bash
  npm install
```

Start the App

```bash
  npm start
```
